# base64_tool
base64 encrypt/decrypt tool

```
Base64 Tool Version 1.1.0
Copyright (c) 2022 by yagas <yagas@sina.com>
::

base64 command input output

Command:
  -c            encode
  -c:safe       safe-encode
  -d            decode

Arguments:
  input         origin file path
  output        target file path

E.g. base64 -d .\base64.txt .\decode.pdf
```
